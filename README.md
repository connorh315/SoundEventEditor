# SoundEventEditor (SEE)

<p align="center">
	<img src="Images/see.png" width="200">
</p>

A fully open-source Sound Event Editor for version 1 SOUND_EVENT files.

SEE will allow you to open, edit, and save SOUND_EVENT files within a GUI. Every file from LEGO Dimensions is supported, and you can even create your own files from scratch.

## Installing

1. Install the .NET Runtime 8 if you don't have it already. You can download it from the [.NET website](https://dotnet.microsoft.com/download/dotnet/8.0).
2. Download the latest release from the [releases page](/releases/)
3. Double-click the `SoundEventEditor.exe` file to run the application.
